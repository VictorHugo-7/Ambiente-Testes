npm init -y                             --> Iniciar um projeto

npm install express                     --> express

npm install mongoose                    --> mongose: manipular os dados no banco

npm install --save-dev nodemon          --> nodemon: restartar sozinho nosso servidor. 
    * colocar no arquivo package.json   --> "dev": "nodemon index.js" | 
    * npm run dev                       --> iniciar o servidor com nodemon

node index.js                           --> Rodar o projeto

--save-dev                              --> Dependência que será instalada só no meu computador. Não vai ir para o servidor as dependência instaladas com o --save-dev

schema                                  --> No mongodb se refere à (estrutura dos meus dados)

async e await                           --> Processo assincrono: espere isso acontecer para depois vc continuar o fluxo da aplicação
 
Códigos HTTP
    - 2xx - Confirmação / Sucesso
        200 - Requisição realizada com sucesso
        201 - Criado - Requisição de criação realizada com sucesso.

    - 4xx - Erro do cliente (front-end)
        - Digitação errada
        400 - Bad Request
        401 - Unauthorized
        403 - Forbidden (Proibido)
        404 - Not Found (Não foi encontrado)
        
    5xx - Erro no servidor (falha ao concluir solicitação)
        500 - Internal Server Error
        502 - Bad Gateway
